# Portafolio personal — Younes Demraoui

Sitio web personal construido con **Astro** y **Tailwind CSS** para presentar perfil profesional, experiencia y proyectos.

## Stack
- Astro 4
- Tailwind CSS 4
- TypeScript

## Estructura del proyecto
```text
/
├── public/
├── src/
│   ├── components/
│   ├── layouts/
│   ├── pages/
│   └── styles/
├── FUTURAS_MEJORAS.md
└── package.json
```

## Scripts
- `npm run dev`: servidor local en desarrollo.
- `npm run build`: validación de tipos y build de producción.
- `npm run preview`: previsualización de build local.
